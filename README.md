# 🏢 Torre Elektra — Sistema de Reservas

Sitio de reservas para la Torre Elektra en PixelPlay Minecraft. 36 pisos, 33x33 bloques por piso, departamentos y oficinas.

## Stack
- Next.js 15 (App Router)
- Vercel Postgres (Neon)
- Tailwind CSS
- TypeScript
